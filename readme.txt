== Description ==

Azia – A modern social media portfolio theme built for Elementor compatibility


== Installation ==

1. Upload the `azia` folder to your `/wp-content/themes/` directory.
2. Activate the theme through the ‘Themes’ menu in WordPress.

== Frequently Asked Questions ==

= How do I customize the theme? =
You can customize the theme through the WordPress Customizer under Appearance → Customize.
